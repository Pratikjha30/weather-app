# 🌤️ Weather App

A simple, responsive Weather Web Application that provides real-time weather information for any city using live API data.

---

## 🚀 Live Demo
👉 https://your-username.github.io/weather-app

---

## 📌 Features
- 🌍 Search weather by city name  
- 🌡️ Real-time temperature updates  
- ☁️ Weather conditions (sunny, cloudy, rain, etc.)  
- 💨 Wind speed information  
- 📱 Fully responsive design (mobile + desktop)

---

## 🛠️ Built With
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- OpenWeather API
